# PropManager

A multi-module property management platform built with the MERN stack.

## Structure
- `client/` — React frontend (Vite)
- `server/` — Node.js / Express backend
- `database/` — MongoDB schema docs & scripts
- `docs/` — Architecture decisions and API references
- `tests/` — Unit, integration, and e2e tests
- `devops/` — Docker, CI/CD, monitoring

## Getting Started
```bash
# Install all dependencies
npm run install:all

# Start dev servers (client + server)
npm run dev
```
